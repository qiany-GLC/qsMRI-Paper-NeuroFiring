# qsMRI-Paper-NeuroFiring

1. About the License. 
Data and code for the paper about qsMRI detecting neuronal firing, shared under the license: Creative Commons Non-Commercial (CC BY-NC).

2. About the Software requirment.
MATLAB version R2021a or later already installed on your computer.

3. How to download the Code and Data. 
First, open and read qsMRI_ReadMe.pdf in the MAIN branch to get the guidelines about running the code and getting the results.
Second, to download the data, switch branch from "main" to "Code-and-data", then download all the files in that branch to one folder in your computer.
