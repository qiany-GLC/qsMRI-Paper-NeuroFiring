# qsMRI-Paper-NeuroFiring

1. About the License
Data and code for the paper about qsMRI detecting neuronal firing, shared under the license: Creative Commons Non-Commercial (CC BY-NC).
Read the file qsMRI_ReadMe.pdf first, then download all the code and data in the branch: code-and-data.

2. How to download the Code and Data
First, open and read qsMRI_ReadMe.pdf in the MAIN branch to get the guide about running the code and getting the results.
Second, to download the data, switch branch from "main" to "code-and-data", then download all the files in the branch "code-and-data".
